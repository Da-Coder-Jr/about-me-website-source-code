Tips, Hacks & Tricks

VSCode Hacks

  • Zen Mode: Hides all distractions in VSCode. Press `Ctrl + K Z` (Windows/Linux) or `Cmd + K Z` (Mac) to enter Zen Mode.
  • Multi-Cursor Magic: Allows you to place multiple cursors in your text. Hold `Alt` (Windows/Linux) or `Option` (Mac) and click in multiple places where you want to place cursors.
  • Quick Rename: Quickly rename all instances of a variable or function. Highlight the variable and press `F2`, then enter the new name and press `Enter`.
  • Command Palette: Provides quick access to all VSCode commands. Press `Ctrl + Shift + P` (Windows/Linux) or `Cmd + Shift + P` (Mac) to open the Command Palette.
  • Integrated Terminal: Opens a terminal right within VSCode. Press `Ctrl + `` (Windows/Linux) or `Cmd + `` (Mac) to open/close the integrated terminal.
  • Live Server: Instantly see changes in your web projects by running a local server. Install the "Live Server" extension from the VSCode marketplace.

Git & GitHub Hacks

  • Speedy Commits: Combines the staging and commit process into a single step. Run `git commit -am "your message"` to add and commit changes in one go.
  • Undo Last Commit: Allows you to undo your last commit while keeping your changes. Run `git reset --soft HEAD~1` to undo the last commit.
  • Keep Your Fork Up to Date: Synchronizes your fork with the upstream repository.
  • Interactive Rebase: Allows you to edit, combine, or delete commits in your history. Run `git rebase -i HEAD~n` where `n` is the number of commits you want to rebase.
  • GitHub CLI: Manage GitHub repositories from your terminal. Install GitHub CLI and authenticate.
